# 绝区零模拟点击抢号脚本

# 使用方法

- 使用 `getmouse.py` 获取鼠标位置
- 修改 `zzzuid.py` 中的 `x_center`, `y_center`, `range_x`, `range_y`为获取到的位置
- 运行 `zzzuid.py`

# 注意

- 需要安装 `Python3.7.0`
- 需要安装 `schedule`
- 需要安装 `pyautogui`
- 需要管理员权限运行脚本, `Win + X` 使用终端管理员, cd进入脚本目录

## 相关命令

```shell
pip install schedule pyautogui
```

```shell
python getmouse.py
```

```shell
python zzzuid.py
```
